# Adunwealth BeautyHome Website

A luxury beauty brand website designed with elegance and femininity in mind.

## Project Structure

- `index.html`: Main homepage.
- `css/styles.css`: Main stylesheet containing all design variables and rules.
- `js/script.js`: JavaScript for interactivity (mobile menu, animations).
- `assets/`: Directory for images and other static assets.

## Design System

- **Colors**:
  - Primary: Blush Pink (#f6dfe7)
  - Secondary: Glossy Black (#000000)
  - Accent: Gold/Bronze
  - Background: White (#ffffff)

- **Typography**:
  - Headings: Playfair Display (Serif)
  - Body: Inter (Sans-serif)

## Features

- Responsive Design
- Glassmorphism Effects
- Smooth Scrolling
- Fade-in Animations
- Mobile Navigation

## How to Run

Simply open `index.html` in any modern web browser.
